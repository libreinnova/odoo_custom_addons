========================
LBR no partner subscribe
========================

|maturity| |badge1| |badge2|

Disable partner subscription in mail thread
-------------------------------------------

This module disables subscription for partners in the next models:

* Sale order *(sale.order)*
* Purchase order *(purchase.order)*
* CRM opportunity *(crm.lead)*
* Project *(project.project)*
* Project task *(project.task)*
* Invoice *(account.invoice)*
