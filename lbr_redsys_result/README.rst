=================
LBR Redsys result
=================

|maturity| |badge1| |badge2|

Payment feedback on OK result
-----------------------------

This module enables payment transaction feedback on get OK result from Redsys.

It listens */payment/redsys/result/* URL to get confirmation and confirm the payment.

The module requires the `Pasarela de pago Redsys (by Tecnativa, OCA) <https://github.com/OCA/l10n-spain/tree/13.0/payment_redsys>`_ module to be installed.
